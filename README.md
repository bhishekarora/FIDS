# FIDS Server setup 
Flight information display system Free 


## Project setup

### BAckend  Server on port env|| 8080
```
cd FIDS Server
npm i 
node server.js
```

### Socket  Server
```
Embedded in Backend server will expose on port 3000
```


### FIDS Client
```
cd FIDS Client
npm i
```
Run `ng serve --port 8081` for a dev server. Navigate to `http://localhost:8081/`.
